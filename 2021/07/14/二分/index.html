二分

发表于 2021-07-14

认识二分

尽管二分查找的基本思想相对简单，但细节可以令人难以招架 … — 高德纳

Although the basic idea of binary search is comparatively straightforward, the details can be surprisingly tricky… — Knuth

二分是一种理解起来非常简单的查找算法，可以将查找的效率降至logN。
但是到具体代码实现上，考虑一下几点

  1. 区间的开闭 [开区间还是闭区间]
  2. 左右端点 [这个端点是和上面区间开闭联系的，一般来讲具体表现为左闭右开，左闭右闭]
  3. 中点是取整还是进一 [在计算中点的时候到底是(left + right) >> 1还是(left + right + 1) >> 1]
  4. 取不取等于 [是大于等于还是小于等于]
  5. 第一个还是最后一个 [找第一个大于目标的位置还是找最后一个大于目标的位置]

各种条件分析

首先要考虑low+ high 溢出的问题。这里简化，暂不讨论。

边界条件

对比前后两者模版区别

前者是左闭右闭，后者是左闭右开。 后者的lo永远不会等于hi，因为最后一个元素是不碰的(index = length)

所以，在while判断时，前者是 lo<=hi.后者是lo<hi

同样，因为闭区间会覆盖每一个元素。所以前者的mid需要+1，-1。而后者因为是开区间 mid不需要-1

关于lo + high 是否再+1

当数组为偶数时，以左闭右闭且length = 4 为例。

mid = low+hi /2 即 （0+ 3）>>>1 = 1，取的是第二个元素，即偏左的

mid = low+hi +1 /2 即 （0+ 3+1）>>>1 = 2，取的是第三个元素，即偏右的

关于查找区间的上下界

左闭右闭为例，左边界开，左端点闭

例如 剑指 Offer 53 - I. 在排序数组中查找数字 I

如果nums[mid] == target的情况下，说明摸到了元素。那么

  • 上界在 mid+1～hi之 间，最后一次左端点是mid，当跳出循环时，最后一次的左端点即是最终左端点
  • 下界在 lo～mid-1 之间，最后一次右端点为mid，当跳出循环时，最后一次的右端点即是最终右端点

最终答案为 右边界 - 左边界 - 1 or 右端点 - 左端点 + 1

开练

只要有序，就要考虑是否可以二分

leetcode 33 搜索旋转排序数组

自己完成时的思路为，先二分判断左右两边是否有序，找到minIndex ( minIndex >0 )，然后看mid是在 0～ minIndex-1 中还是 minIndex ～length -1 中。进行二分。逻辑是可以的，时间复杂度也达标，但是代码量比较大，写了100来行。不够优雅。看了官方解题后，学习到思路如下

对数组进行二分，一定有一部分是有序的，如果target刚好在这个有序的范围内，ok继续对有序范围进行二分。否则，就是在另外一堆。对另外一堆进行二分

leetcode 34 在排序数组中查找元素的第一个和最后一个位置

也就是前面所说上下界的问题

leetcode 35 搜索插入位置

先二分查，如果查到就返回，如果没有查到，注意此时已经跳出循环lo是>hi的。所以返回hi+1

leetcode 74 搜索二维矩阵

这道题我是一行一行二分的。时间复杂度是m * logN。实际上没必要，当作一个折叠成二维数组的一维数组就好。

leetcode 81 搜索旋转排序数组 II

这道题和33题是姊妹题。

这是 搜索旋转排序数组 的延伸题目，本题中的 nums 可能包含重复元素。
这会影响到程序的时间复杂度吗？会有怎样的影响，为什么？

区别在于这道题中可能出现二分后，左右两端都先等的情况。

比如[3,1,2,3,3,3,3]

左边是 [3,1,2,3] 右边是 [3,3,3]。无法确定夹在左右哪边。官方解题为左边界加一，右边界减一。我认为可以再对两边进行递归。

leetcode 153 寻找旋转排序数组中的最小值

我的想法为其实就是二分到两端都有序，然后比较两个起点的大小。否则的话，一段有序，另一段无序，起点一定在无序那段，对无序那段再二分。官方题解的关注点在于，中点和左右两点的比较。也可以理解为有序无序。中点比hi小，那么左侧有序，最小值可能是mid也可能在mid左边，所以可以是hi = mid。否则右侧无序 lo = mid+1

leetcode 69 x的平方根

朴素的思想是

while ((result + 1) * (result + 1) <= x) { result++; }

可以用0为下界，x/2为上界来进行二分
